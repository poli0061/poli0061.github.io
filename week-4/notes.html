The first one tag - "< a >< /a >" which is used for the links. This is a paired tag,that is usually added a link between the tags and in addition we should to add href, which will link to the added material.

The second tag - "< form >< /form >" which is used for the forms. Inside the tag we add information that will allow the user to input data.

The trird tag - "< table >< /table >" which is used for tables. Information is added between tags, tables are useful for organized moments, for example, when you need to organize the hours of work in the week.

The fourth tag - "< tr >< /tr >" which is used for table rows. This tag added between < table > < /table > tag.

The fith tag - "< th >< /th >" which is used for data cell. This tag added between < table > < /table > tag. Used to distribute information across a table.
